Fetch API

Fetch API veb serverə HTTP sorğuları (məsələn, GET, POST, PUT və Delete) etməyə imkan verən xüsusiyyətdir. O, müasir brauzerlərdə quraşdırılıb.Burada siz şəbəkə sorğularıda edə bilərsiniz. Məsələn, şəbəkə sorğusu dedikdə:
Sifariş vermək,
İstifadəçi məlumatlarını yükləmək,
Serverdən ən son yeniləmələri almaq,… və s. nəzərdə tutulur …Və bütün bunlar səhifəni yenidən yükləmədən! Sadə dillə desək, Fetch API vebsaytdan məlumat əldə etməyi və brauzerinizdə həmin məlumatlarla hansısa işi görməyi asanlaşdırır. Məsələn, siz veb saytdan HTML sənədini tələb etmək üçün Fetch API-dən istifadə edə bilərsiniz.

Əsas sintaksis belədir:
let promise = fetch(url, [options])
Url – daxil olmaq üçün adres.
Seçimlər(options) – istəyə bağlı parametrlər: metod, başlıqlar və s.

Axios nədir?

Axios üçüncü tərəf JavaScript kitabxanasıdır. Axios brauzerdə həm Node.js, həm də JavaScript-i dəstəkləyir. Axios, HTTP sorğularını göndərmək üçün istifadə edilən vəd(promise) edilmiş JavaScript kitabxanasıdır. Bunu JavaScript-in dogma(native) fetch() funksiyasına alternativ kimi düşünə bilərsiniz. Bu sorğuların verilməsi və cavabların idarə edilməsi prosesini asanlaşdırır. O, həmçinin avtomatik JSON təhlili kimi xüsusiyyətləri təmin edir.
import axios from 'axios';
async function fetchData() {
try {
const response = await axios.get("https://api.example.com/users")
setUser(response.data)
} catch (error) {
console.error(error);
} }

Fetch və ya Axios ilə Async/Await: Siz asinxron əməliyyatlarla işləyərkən kodunuzu daha oxunaqlı etmək üçün müasir async/await sintaksisindən istifadə edə bilərsiniz. Bu, həm Fetch API, həm də Axios ilə edilə bilər.

React’s useState Hook

useState, funksional komponentə vəziyyət əlavə etməyə imkan verən React Hook-dur. O, iki dəyəri olan massivi qaytarır:dəyəri və onu yeniləmək üçün funksiya. Hook arqument kimi ilkin vəziyyət dəyərini alır və təyinedici funksiyası çağırıldıqda yenilənmiş vəziyyət dəyərini qaytarır. React-də useState istənilən növ dəyəri saxlaya bilər. Buraya string, number və Boolean kimi primitiv məlumat növləri, həmçinin massiv, obyekt və funksiya kimi mürəkkəb məlumat növləri daxildir. Əsasən, JavaScript dəyişənində saxlanıla bilən hər şey useState tərəfindən idarə olunan vəziyyətdə saxlanıla bilər.
const [state, setState] = useState(initialValue);
// Objects
const [state, setState] = useState({ name: 'John', age: 30 });

const updateName = () => {
setState({ ...state, name: 'Jane' });
};
const updateAge = () => {
setState({ ...state, age: state.age + 1 });
};
// Arrays
const [array, setArray] = useState([1, 2, 3, 4, 5]);
const addItem = () => {
setArray([...array, 6]);
};
const removeItem = () => {
setArray(array.slice(0, array.length - 1));
};
